# k8s-learn

## Proposito del repositorio

Este repositorio nace como un espacio de estudio para almacenar y organizar informacion tecnica relacionada con Kubernetes, con enfasis en la comprension de su codigo fuente.

Su objetivo principal es reunir material didactico y colaborativo en Español que ayude a entusiastas y profesionales a fortalecer su aprendizaje, profundizar en conceptos clave y mejorar su dominio practico del ecosistema Kubernetes.

## Contexto de creacion

El proyecto fue creado para dar continuidad a sesiones de estudio y documentar hallazgos de forma ordenada. La idea es construir una base de conocimiento compartida, accesible y evolutiva, donde se registren notas, resumenes y recursos que faciliten el aprendizaje progresivo del proyecto Kubernetes.

## A quien esta dirigido

- Personas interesadas en aprender Kubernetes desde una perspectiva tecnica.
- Equipos o grupos de estudio que quieran colaborar en Español.
- Profesionales que busquen entender mejor el funcionamiento interno de Kubernetes.
